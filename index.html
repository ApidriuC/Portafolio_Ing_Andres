HABILIDADES Y CONOCIMIENTOS

Me destacó por ser una persona autodidacta, que aprende rápido y maneja situaciones bajo presión. Soy una persona que no para hasta lograr sus objetivos y que siempre quiere ser el mejor en lo que hace.

Cuento con conocimiento en las siguientes tecnologías y lenguajes de programación que utilizo comúnmente para la elaboración de soluciones:

Tecnologías Principales

Java

Java es un lenguaje de programación sencillo y orientado a objetos, este lenguaje es el más utilizado en toda mi trayectoria y en el que más tengo experiencia.

JavaScript

JavaScript es un lenguaje de programación interpretado, con este lenguaje he realizado la mayoría de mis proyectos grandes durante mi vida Universitaria.

C#

C# es un lenguaje de programación orientado a objetos y a componentes, este lenguaje cuenta con una sintaxis muy parecida a Java por lo que se me ha facilitado la compresión del mismo y la mejora constante.

Python

Python es uno de los lenguajes de programación más utilizados en la actualidad gracias a su facilidad de uso y compresión. Adicionalmente por su facilidad en el manejo de redes neuronales, maching-learning.

Tecnologías Secundarias

React

React es una herramienta de Frontend muy útil, me siento muy comodo utilizando esta libreria de JavaScript y cada vez aprendo más sobre ella.

MySQL

MySQL es un sistema de gestión de base de datos, no me considero un DBA pero sin duda esta es una herramienta muy útil y la que más he utilziado a lo largo de mi trayectoria.

Otros Conocimientos

React Native

Hice uso de React Native para el desarrolo de un proyecto para móviles en SO Andorid, me gustó mucho trabajar con esta tecnología.

Flutter

Utilice esta tecnología para el desarrollo de un aplicativo móvil y me llamó mucho la atención la facilidad para desarrollar interfaces de usuario.

Oracle

He usado poco esta tecnología pero gracias al conocimiento que tengo de otros lenguajes SQL he podido llegar a usar esta herramienta para ciertos proyectos.

Express/NodeJS

Esta solucion tecnológica me llamó mucho la atención gracias a su escalabilidad y sus modulos de desarrollo, me gustaría seguir profundizando en esta tecnología.
